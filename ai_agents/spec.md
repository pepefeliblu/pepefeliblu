# Project Constraints

## Audience
Recruiters, software engineers, and tech readers.

## Tone
Clean, professional, minimal, opinionated but not flashy.

## Hard Constraints
- Content must be **Markdown-first** (editable in Obsidian, VS Code, or GitHub web UI)
- No CMS, no backend, no database
- No unnecessary libraries, analytics, or ads
- Keep everything lightweight and dependency-minimal
- Only add React components when strictly necessary (currently used for seasonal effects only)
- Favor clarity, simplicity, and long-term maintainability

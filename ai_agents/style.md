# Design Philosophy: Digital Moleskine

The site captures the feeling of a physical Moleskine notebook: dot grid background, creamy paper tones, soft charcoal ink, burnt orange accents.

- **Minimalist & Modern**: No skeuomorphic textures. Solid colors and CSS patterns only.
- **Content-first**: Like writing in a fresh notebook. The whole screen is the "page".
- **Spacing**: Generous margins, vertical rhythm in multiples of 4px/8px.

All color values, fonts, and implementation details live in `src/styles/global.css` as CSS custom properties. That file is the source of truth for any visual changes.

# dev-rueda

Personal website and tech blog built with [Astro](https://astro.build).

## Features

- **Markdown-first** blogging
- Clean, minimal design
- High performance (static generation)
- Dark/Light mode support
- Markdown CV support

## Setup

1. **Install dependencies**:
   ```bash
   pnpm install
   ```

2. **Run locally**:
   ```bash
   pnpm dev
   ```

3. **Build for production**:
   ```bash
   pnpm build
   ```

## Adding Content

- **Blog Posts**: Add new `.md` files to `src/content/blog/`.
- **CV**: Edit `src/pages/cv.md`.

## Deployment

Deployment is handled automatically via GitHub Actions when pushing to `main`.
